Welcome to Social Square!

Social Square™은 가장 빠르고 , 간편하게 여러분을 관리할수있는 툴이래요





아래 가이드를 선택해서 가이드를 볼수있습니다





메모 가이드

간단한 메모를 편집할수있는기능입니다.

data가 localstorage 에 저장되기 때문에 , 캐시 / 쿠키를 삭제해도 데이터가 보존되며 , 데이터 종범에 대해서 개발자는 책임지지 않습니다.

텍스트 작성부분에 원하는 텍스트를 쓴뒤 세이브를 눌러주세요.

수정버튼을 통해 수정기능도 지원합니다.

트윗가이드

원하는 사용자의 유저 타임라인과 그 사람에게 온 멘션을 조회할수있습니다.

세팅메뉴를 통해 ID를 입력하면 볼수있습니다

실시간 검색어

네이버의 실시간검색어입니다. 1분마다 갱신됩니다.

RSS피드 가이드

RSS피드를 볼수있는곳입니다. 텍스트입력창에 RSS를 누르고 + 를 누르면 RSS가 표시됩니다.

옆의 버튼을 클릭해 연합뉴스의 소식을 볼수있습니다.

날씨 가이드

weather underground 의 API 를 이용하여 날씨를 볼수있는 기능입니다.

가끔 api 리밋이 걸리면 날씨가 안보일때도 있습니다.
